Возможные типы элемента input

Возможные типы элемента input
typeВид в браузереОписание
buttonКнопка.
checkboxФлажки. Позволяют выбрать более одного варианта из предложенных.
fileПоле для ввода имени файла, который пересылается на сервер.
hidden (не отображается)Скрытое поле. Никак не отображается на веб-странице.
imageПоле с изображением. При нажатии на рисунок данные формы отправляются на сервер.
passwordОбычное текстовое поле, но отличается тем, что все символы показываются звёздочками.
radio Переключатели. Один вариант из нескольких.
resetКнопка для возврата данных формы в первоначальное значение.
submitКнопка для отправки данных формы на сервер.
textТекстовое поле. Ввод символов с помощью клавиатуры.
colorВиджет для выбора цвета.
dateПоле для выбора календарной даты.
datetimeУказание даты и времени.
datetime-localУказание местной даты и времени.
emailДля адресов электронной почты.
numberВвод чисел.
rangeПолзунок для выбора чисел в указанном диапазоне.
searchПоле для поиска.
telДля телефонных номеров.
timeДля времени.
urlДля веб-адресов.
monthВыбор месяца.
weekВыбор недели.
