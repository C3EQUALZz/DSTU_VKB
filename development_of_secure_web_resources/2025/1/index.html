Лабораторная работа №1. Знакомство с HTML

Дисциплина: Разработка защищённых WEB-ресурсов. Содержание выполненных заданий:

  1. Простая веб-страница — теги h1, p, hr, title.
  2. Форматирование текста — b, strong, i, em, u, small, mark, del, ins, sub/sup.
  3. Списки — ul, ol, вложенные, разные маркеры.
  4. Ссылки — внешняя, якорная, скачивание файла.
  5. Изображения — img, размеры, относительные пути, изображение-ссылка.
  6. Таблицы — table, caption, colspan/rowspan.
  7. Тег input — все типы атрибута type.
  8. Формы — form, input, textarea, select, чекбоксы, радиокнопки.
  9. Семантическая структура — header, nav, main, article, aside, footer.
  10. Аудио и видео — теги audio, video, iframe YouTube/Vimeo.
  11. Iframe — встраивание Google Maps.

Все страницы — на русском, валидный HTML5. Подробное описание — в README.md.
