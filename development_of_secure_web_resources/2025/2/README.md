# Лабораторная работа №2. Стилевое оформление WEB-страниц

**Дисциплина:** Разработка защищённых WEB-ресурсов
**Тема:** возможности CSS по графическому оформлению элементов страницы.

## Цель работы

Освоить основные возможности CSS: способы подключения стилей, базовые
группы селекторов (элемента, класса, идентификатора, потомка, атрибута,
псевдокласса, псевдоэлемента), стилизацию ссылок-кнопок и таблиц,
оформление контентных блоков, форм и навигационных меню, базовую
адаптивность через `@media`.

## Краткая теория

### Способы подключения CSS

1. **Внешние стили** (рекомендуется):
   ```html
   <link rel="stylesheet" href="css/styles.css">
   ```
2. **Внутренние** — `<style>...</style>` в `<head>`.
3. **Инлайн** — атрибут `style="..."` непосредственно у элемента
   (используем минимально, только когда CSS-файл избыточен).

В этой лабе используется *внешний* способ: общие правила вынесены в
`css/common.css`, у каждого задания — собственный CSS-файл.

### Основные группы селекторов

| Группа | Пример | Что выбирает |
|--------|--------|---------------|
| Элемент | `p { ... }` | Все `<p>` |
| Класс | `.lead { ... }` | Элементы с `class="lead"` |
| Идентификатор | `#article` | Элемент с `id="article"` |
| Потомок | `.card h3` | `<h3>` внутри `.card` |
| Атрибут | `a[target="_blank"]` | Ссылки с `target="_blank"` |
| Псевдокласс | `a:hover`, `:nth-child` | По состоянию/позиции |
| Псевдоэлемент | `::first-letter`, `::after` | Виртуальный фрагмент элемента |

## Состав работы (6 заданий)

### Задание 1. Все типы CSS-селекторов

Файлы: [`01-selectors.html`](01-selectors.html) + [`css/01-selectors.css`](css/01-selectors.css).
В CSS реализованы по очереди **все семь** групп селекторов. Каждое
правило применяется к конкретному видимому элементу: ссылке, абзацу,
списку, заголовку — и его эффект заметен невооружённым глазом.

### Задание 2. Стилизованные ссылки-кнопки

Файлы: [`02-buttons.html`](02-buttons.html) + [`css/02-buttons.css`](css/02-buttons.css).
Три варианта:
- `.btn-primary` — фиолетовый фон, контрастный белый текст, `:hover`
  меняет цвет, `:active` вдавливает кнопку.
- `.btn-shadow` — оранжевый градиент, мощная `box-shadow` (как в
  примере PDF — `2px 2px 15px 0 rgba(255,151,32,.863)`), увеличение
  `scale(1.05)` при наведении, стрелка через `::after`.
- `.btn-ghost` — контурный вариант с подчёркиванием через `::before`,
  при `:hover` заливается полностью.

### Задание 3. Таблица 7×4

Файлы: [`03-table.html`](03-table.html) + [`css/03-table.css`](css/03-table.css).
Таблица «Список сотрудников отдела»:
- стилизованные заголовки столбцов (`thead th`) — градиент;
- стилизованные заголовки строк (`tbody th`) — светлый фон, контур
  справа;
- чередующиеся `:nth-child(odd)` / `:nth-child(even)`;
- разделители столбцов через `td + td { border-left: ... }`;
- курсор `cursor: pointer` и подсветка строки при `:hover`;
- скруглённые углы (через `border-radius` + `overflow: hidden`)
  и `box-shadow`.

### Задание 4. Информационный блок Дзен-стиля + страница статьи

Файлы:
- [`04-info-block.html`](04-info-block.html) + [`css/04-info-block.css`](css/04-info-block.css) — карточка;
- [`04-article.html`](04-article.html) + [`css/04-article.css`](css/04-article.css) — статья.

Карточка содержит изображение (inline SVG-горный пейзаж), аватар
автора, заголовок и краткое описание, ссылку «Читать полностью».
На клике переход на страницу статьи, где использована своя
типографика (Georgia, увеличенный line-height) и стилевое
выделение цитат.

### Задание 5. Форма регистрации (адаптивная)

Файлы: [`05-registration.html`](05-registration.html) + [`css/05-registration.css`](css/05-registration.css).
- Поля: имя, фамилия, email, пароль, подтверждение пароля.
- Кнопки: «Регистрация», «Войти», «Забыли пароль?».
- Обязательные поля помечены красной звёздочкой, поведение
  валидации — через `:invalid:not(:placeholder-shown)`.
- Плавные `transition` на фокусе.
- Градиентный фон на всём экране.
- Адаптивность: при ширине < 560 px поля «Имя/Фамилия» и
  «Пароль/Подтверждение» становятся в один столбец.

### Задание 6. Вертикальное + горизонтальное меню

Файлы: [`06-menu.html`](06-menu.html) + [`css/06-menu.css`](css/06-menu.css).
- Слева — **вертикальное** меню, `position: fixed`, ширина 200 px,
  высота 100vh. Подсветка пункта по `:hover` и заранее выделенный
  «активный» пункт.
- Сверху — **горизонтальное** меню (sticky-header), CTA-кнопка справа.
- На экранах < 720 px вертикальное меню превращается в горизонтальную
  ленту наверху страницы.

## Как запустить

### Двойной клик
Просто открыть `index.html` в любом современном браузере.

### Локальный сервер
```bash
cd development_of_secure_web_resources/2025/2
python3 -m http.server 8000
```
В браузере — <http://localhost:8000/>.

## Проверка (выполнялась)

```bash
python3 -m http.server 8766 --bind 127.0.0.1 &
for f in index 01-selectors 02-buttons 03-table 04-info-block \
         04-article 05-registration 06-menu; do
    curl -s -o /dev/null -w "%{http_code} $f.html\n" \
         http://127.0.0.1:8766/$f.html
done
```
Все 8 страниц и все 8 CSS-файлов возвращают `200 OK`.
