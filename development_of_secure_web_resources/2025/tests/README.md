# Playwright smoke-тесты для лабораторных работ

Набор end-to-end smoke-тестов на Playwright для всех 5 лабораторных
работ дисциплины «Разработка защищённых WEB-ресурсов» (`2025/1..5`).
22 теста, ~5 секунд.

## Что проверяется

| Лаба | Что в smoke |
|------|--------------|
| **ЛР1** | `index.html` содержит ровно 11 ссылок на задания; страница 01 имеет заголовок и `<hr>`; таблица типов `input` ≥20 строк. |
| **ЛР2** | Таблица 7×4 содержит 7 строк; на `:hover` кнопки меняется фон (computed style до/после); все обязательные поля формы регистрации видны. |
| **ЛР3** | Grid карточек переключается в 4/3/2 столбца при ширине 1280/900/480 px (через computed style); CSS-only гамбургер на 360 px раскрывает меню по клику на `<label>`. |
| **ЛР4** | Задача 1 (погода) — корректные ответы на ввод `-10` и `25`; задача 5 — максимум массива через prompt; задача 8 — возраст 25 → доступ разрешён; задача 10 — гласные в "Привет" = 2. |
| **ЛР5** | Модалка открывается; невалидный email подсвечивает ошибку; валидный submit показывает «Спасибо!»; таймер реально отсчитывает секунду; галерея показывает превью на hover; задание 7 — старт 7/0, добавление студента → 8. |

## Запуск

```bash
cd development_of_secure_web_resources/2025/tests

# одноразовая установка (если ещё не делали)
npm install
npx playwright install chromium

# прогон всех smoke-тестов
npm test
# или то же самое:
npx playwright test
```

Playwright сам поднимет `python3 -m http.server 8800` в каталоге
`2025/` и остановит его после прогона (см. `webServer` в
`playwright.config.ts`).

## Полезные команды

```bash
# С подробным выводом
npx playwright test --reporter=list

# Только тесты ЛР5
npx playwright test -g "ЛР5"

# Один тест по совпадению имени
npx playwright test -g "гамбургер"

# Открыть HTML-отчёт после прогона
npx playwright show-report
```
