# 10 лабораторная работа (openssl)

Скачайте `openssl` и настройте, чтобы он был в переменной пути `PATH`. 
В моем случае это давно уже было настроено, так что мануала не будет по настройке.

Для того чтобы узнать версию, нужно использовать команду: 

```bash
openssl version
```

Для того чтобы узнать более подробную информацию, нужно использовать команду:

```bash
openssl version -a
```

Для вывода доступных команд можно использовать команду, которая представлена ниже:

```bash
openssl help
```

Вывод доступных команд с подкомандами:

```bash
openssl dgst -h
```

Вывод всех доступных шифров:

```bash
openssl ciphers
```

Измерение производительности:

```bash
openssl speed
```

Измерение производительности сети:

```bash
openssl s_time -connect remove.host:443
```
