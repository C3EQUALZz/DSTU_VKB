# Задание 8. 

## Условие

Выполните шифрование по алгоритмам DES-EDE и 3DES, используя только функцию DES, сравните свои результаты с результатами реализаций в
OpenSSL, убедитесь в их совпадении (например, шифруйте вручную, а расшифровывайте OpenSSL), каждый раз при шифровании запоминайте
время.

### Подготовка

1. Убедитесь, что OpenSSL установлен и доступен в PATH
2. Создайте тестовый файл для шифрования (например, `plain.txt`)

Для автоматизации процесса можно использовать PowerShell скрипт `des3-automation.ps1` (см. файл в этой директории).

```powershell
.\des3-automation.ps1 -InputFile "plain.txt"
```
