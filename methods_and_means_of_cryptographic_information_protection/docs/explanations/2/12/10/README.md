# Задание 10. 

## Условие

Сформировать самоподписаный сертификат. 

## Практическая реализация

Для формирования сертификата можно использовать команду: 

```bash
openssl req -new -x509 -days 365 -nodes -out D:\PycharmProjects\DSTU_VKB\methods_and_means_of_cryptographic_information_protection\docs\explanations\2\12\10\cert.pem -keyout D:\PycharmProjects\DSTU_VKB\methods_and_means_of_cryptographic_information_protection\docs\explanations\2\12\10\cert.key 
```

> [!IMPORTANT]
> У вас совершенно иные пути могут быть.
