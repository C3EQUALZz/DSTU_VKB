## Задание 2.2.

Найти в интервале (1000, 1000 + 300) все простые числа. Пусть L(i) — разность между двумя соседними простыми
числами. Построить гистограмму для £(/). Вычислить вы
борочное среднее £сред.. Сравнить с величиной 1п(х), где х —
середина интервала. Задано: количество простых чисел для
деления 5—2 0 , количество оснований 1—3.

### Выполнение

Для запуска выполнить команду, которая представлена ниже

```bash
python -m cryptography_methods.cli primality-check find-in-interval -tp 5 -fb 1
```

> [!NOTE]
> `-tp` - количество простых чисел для деления (5-20)
> `-fb` - количество оснований (1-3)
