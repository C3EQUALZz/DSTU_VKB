### Установка и запуск

Создайте `.venv`, используя `Pycharm` или командной строки. Вся информация есть в Интернете. 

> [!IMPORTANT]
> Перезапустите терминал, чтобы появился `.venv`

Дальше устанавливаем `uv`:

```bash
pip install uv
```

Устанавливаем все зависимости с помощью `uv`:

```bash
uv sync
```
