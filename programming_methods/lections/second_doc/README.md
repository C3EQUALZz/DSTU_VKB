### Метод скользящего окна

Метод скользящего окна используется для поиска наибольшей подстроки или подмассива, обладающего каким-то свойством.
Алгоритм работает за O(n)

Тут есть такие пункты в алгоритмах:

1. Расширяем подстроку: Увеличиваем right, пока в подстроке нет повторяющихся символов.
2. Сужаем подстроку: Увеличиваем left, пока в подстроке есть повторяющиеся символы.
3. Повторяем действия с П.1.
