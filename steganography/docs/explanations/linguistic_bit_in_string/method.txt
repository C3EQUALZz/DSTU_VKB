МЕТОД СОКРЫТИЯ ОДНОГО БИТА В ТЕКСТОВОЙ СТРОКЕ
================================================

Лингвистический параметр: чётность числа русских гласных букв в строке.

Множество гласных русского алфавита (в обоих регистрах):
    {А, Е, Ё, И, О, У, Ы, Э, Ю, Я}.

Алгоритм классификации произвольной строки S:
    1. Пусть V(S) = количество символов S, принадлежащих множеству
       гласных русских букв (без учёта регистра).
    2. Если V(S) mod 2 == 0  → S входит в множество Y (бит «1»),
       программа возвращает ответ «ДА».
    3. Если V(S) mod 2 == 1  → S входит в множество N (бит «0»),
       программа возвращает ответ «НЕТ».

Любая строка естественного русского языка (включая строки с ошибками,
строки длиной до 50 символов и пр.) однозначно попадает в Y или N,
поэтому метод определён на всём множестве предложений.

Чтобы передать нужный бит, отправитель составляет осмысленную фразу
с подходящей чётностью числа гласных. Чётность можно при необходимости
скорректировать одной правкой: заменить слово синонимом с другим
числом гласных, добавить или убрать частицу («не», «же», «и»,
«а» и т.п.), переставить слово.

Преимущества метода:
  - применим к любой строке: и Y, и N покрывают всё множество предложений;
  - не требует управляющих символов, не нарушает читаемость;
  - устойчив к малозаметным правкам пунктуации и регистра.

Ограничения:
  - один бит на строку;
  - не работает для строк без русских букв (V = 0 даёт «ДА» по соглашению).
