# Теория псевдослучайных генераторов

## Установка

Создайте `.venv` с помощью `Pycharm` / `uv` / `python`. 

> [!IMPORTANT]
> Ожидается, что `.venv` будет с `Python 3.12`.

После создания `.venv` убедитесь, что у вас терминале применилось виртуальное окружение. 
Остается дело за малым: 

- Ставим `uv`:

  ```bash
  pip install uv
  ```

- Ставим `Python` проект, используя команду ниже:

  ```bash
  uv pip install -e ".[dev]"
  ```

Лабораторные работы готовы к использованию!
